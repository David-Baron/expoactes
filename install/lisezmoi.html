Installation :

Avant d'installer :

Le logiciel ne fonctionne que dans un environnement PHP + MySQL.  Vous devez donc disposer d'un hébergement qui offre ces fonctionnalités. Certains hébergements liés à un abonnement Internet fournissent cet environnement inclus dans le prix de votre connexion (par exemple Free.fr ).
Il existe aussi de nombreux hébergeurs payants qui, pour quelques euros par mois fournissent un hébergement solide et exempt de toute publicité. Par exemple  : One.com,  Nuxit, Hosteur, PhpNet, Yellis, OVH, ... proposent des hébergements PHP + MySQL.

ATTENTION : Expoactes 3 nécessite au moins les versions 5.0.1 de PHP et 4.0.1 de MySQL.

Vérifiez bien que votre hébergeur propose ces versions ou des versions supérieures (ce qui est assez courant car les versions les plus récentes sont actuellement des versions 5.x).

Les explications données ici font l'hypothèse que votre ordinateur fonctionne sous Windows.

Si vous souhaitez préalablement tester ExpoActes sans disposer d'hébergement, il est possible de le faire fonctionner en mode local après avoir installé le logiciel gratuit EasyPHP ou WampServer, lisez alors les instructions spécifiques.

Préparation de l'installation :

Le logiciel ExpoActes doit être déployé sur le serveur de l'hébergeur. Bien souvent, on ne dispose que de droits assez limités pour travailler directement sur ce serveur. Il convient donc de préparer l'installation sur votre ordinateur local puis ensuite de transférer le site sur le serveur de l'hébergeur.

Pour cela, nous allons procéder comme suit :

  1. Créer une "copie maître" du site sur votre propre ordinateur;
  2. Transférer avec FTP le site maître sur le serveur de l'hébergeur;
  3. Lancer l'installation de la base de données sur le serveur MySQL de l'hébergeur et y placer les premières données.

Au terme de ces trois phases, l'application est prête à recevoir vos données et bientôt vos visiteurs.

Il est aussi possible à ce moment de peaufiner les réglages du site en configurant d'autres paramètres facultatifs.

Pour résoudre les problèmes qui pourraient se poser, consulter la liste des codes d'erreur et des pistes de solution.

Création du "site maître"

Le programme d'installation expoactes_install_xxxx.exe crée automatiquement tous les fichiers nécessaires.  

Vous devez d'abord préciser le répertoire de votre ordinateur dans lequel vous souhaitez installer la copie-maître.

REMARQUE : Si vous avez déjà réalisé l'installation mais que vous souhaitez changer le répertoire de la copie-maître du site, il est préférable de procéder comme suit :
  1. Faire une nouvelle installation en désignant le répertoire de votre choix;
  2. Si votre copie maître est déjà configurée, copier tous les fichiers de l'ancien répertoire _config vers le nouveau;
  3. Supprimer l'ancien répertoire SANS utiliser le programme de désinstallation. De cette façon,  l'ordinateur retrouvera automatiquement le nouvel emplacement lors des mises à jour suivantes.


Les différents éléments du logiciel sont classés dans plusieurs sous-répertoires.

Parmi ceux-ci le répertoire _config est particulier car il contient les éléments spécifiques à VOTRE site qui ne peuvent être stockés dans la base de données.  Les autres répertoires contiennent essentiellement des programmes PHP. S'agissant d'un logiciel "open source", vous avez la possibilité de modifier ces programmes à votre guise pour les adapter à vos besoins spécifiques.

Toutefois, si vous souhaitez bénéficier des mises-à-jour du logiciel, il vaut mieux ne rien modifier dans ces programmes car les mises-à-jour supprimeront irrémédiablement vos adaptations personnelles.  Elles ne remplaceront par contre pas les fichiers situés dans le répertoire _config.

Il faut alors choisir le type d'installation car le même programme est aussi capable de mettre à jour une version précédente du logiciel. Il s'agit ici d'une "Nouvelle installation".

Vous devez ensuite préciser les codes d'accès à votre base MySQL. Ces valeurs doivent vous avoir été communiquées par votre hébergeur. Le cas échéant vous devrez aussi préciser l'identification de votre base EasyPHP.

Ces données étant précisées, le programme d'installation crée les répertoires nécessaires sur VOTRE ordinateur et y copie les fichiers adéquats.

La "copie maître" du site est à présent prête. Elle pourra, par la suite, être modifiée par d'autres actions de configuration mais dans un premier temps, il vaut mieux la laisser telle qu'elle a été générée.

Transfert du site maître sur le serveur de l'hébergeur

Pour ce transfert, vous devez utiliser un client FTP (si vous n'en avez pas, vous pouvez télécharger FileZilla qui est un logiciel libre). L'adresse du serveur FTP de l'hébergement et les codes de login et mot de passe vous ont été transmis par l'hébergeur.

Le module ExpoActes peut être placé n'importe où dans l'arborescence de votre site. Toutefois, nous vous conseillons de créer un répertoire "actes" dès la racine du site et de transférer l'ensemble des répertoires et fichiers de votre "site maître" dans ce répertoire "actes".

Création de la base de données et initialisation du site

Lorsque tout le site maître a été transféré, il faut parachever l'installation sur le serveur de l'hébergeur en initialisant la base de données et en créant le premier code d'accès.

Si PHP fonctionne correctement, l'accès à la page principale (http://<URL DE VOTRE SITE>/actes/index.php) va vous conduire automatiquement dans la procédure d'installation.
Pour cela tapez dans le navigateur l'adresse de votre site suivie de actes/index.php

Le programme vérifie d'abord que votre hébergement est compatible et qu'il fonctionne bien. Le cas échéant des messages d'erreurs sont affichés et des propositions d'actions sont énumérées dans une page d'aide spécifique.

Si tout est "OK", l'écran demande de donner les informations pour créer votre compte d'administrateur.  Le nom est obligatoire ainsi que le code de login et les deux copies du mot de passe.  Ce dernier doit comporter entre 6 et 15 caractères et il est conseillé d'y inclure au moins des lettres et des chiffres.  

On ne saurait trop insister sur l'importance de ce mot de passe car celui qui parviendrait à le deviner pourrait TOUT faire sur la base de données, y compris récupérer toutes les données pour lui et/ou tout supprimer ! Il est donc sage de prévoir au moins 10 caractères alternant lettres, chiffres et même signes particuliers (-=+?...).

Dès que ces données sont introduites,  il suffit de presser le bouton "ENVOYER" pour que la base se crée, que la liste des prénoms féminins soit constituée (utile pour la lecture des mariages au format CSV uniquement) et que votre compte d'administrateur soit enregistré.

Cela fait, il ne reste plus qu'à charger des données ...

Pour plus de détails sur les fonctions du logiciel, consultez les fichiers d'aide.

Dès que vous êtes familier avec l'utilisation de ExpoActes, vous pouvez aussi adapter certains comportements grâce à paramétrage du logiciel.
