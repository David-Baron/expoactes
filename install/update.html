Mise à jour de ExpoActes


Vous venez de terminer la première partie de la mise à jour en modifiant la "copie maître" de ExpoActes qui est sur votre ordinateur.


Il reste à appliquer les modifications INDISPENSABLES à votre base de données hébergée :

  1. Vous devez d'abord copier l'ensemble des fichiers sur votre hébergement à l'aide de votre logiciel FTP. (Cette opération est inutile si vous testez le logiciel en local avec EasyPHP ou WampServer)

  2. Avec votre navigateur internet,  visitez la page d'accueil du site qui vous renverra automatiquement vers la procédure de post-installation. Ce programme vous demandera un login et mot de passe de niveau "administrateur" (sauf si vous êtres déjà connecté ainsi). Il détectera ensuite les changements nécessaires sur votre base de données MySQL et les effectuera automatiquement lorsque cela est nécessaire.

  3. A partir de la version 3.2, le codage utilisé par ExpoActes est désormais UTF-8 qui est le standard le plus courant aujourd'hui. Toutefois, il ne nous est pas possible de modifier quelques fichiers que vous avez peut-être adapté vous-même à vos besoins. C'est notamment le cas des fichiers présents dans le répertoire _config et en particulier :
    • bandeau.htm
    • commentaire.htm
    Aussi, si le bandeau supérieur du site comporte des lettres accentuées mal affichées (remplacées par des symboles bizarres), vous devez modifier manuellement ces fichiers et remplacer les caractères mal affichés par des entités HTML. Par exemple, le lettre "é" sera remplacée par "&eacute;"
    Une liste des entités HTML est disponible ici.
